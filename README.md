# SafeLoad AI

SafeLoad AI is building the missing layer in freight security — real-time custody verification that closes the trust gap left by existing identity and visibility tools.

**Live site:** https://safeload-website.onrender.com/

## About

This repository contains the marketing/informational website for SafeLoad AI, covering:

- **Home** — overview of the product and mission
- **About Us** — the custody verification gap SafeLoad addresses and why it matters
- **Our Solution** — how the SafeLoad app verifies custody at each step of a shipment
- **Join Pilot Program** — sign-up for early pilot partners

## Tech Stack

- [React](https://react.dev/) (Create React App)
- [React Router](https://reactrouter.com/) for client-side routing
- [Framer Motion](https://www.framer.com/motion/) for animations

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm start
```

The app will be available at `http://localhost:3000`.

## Build

```bash
npm run build
```

This produces an optimized production build in the `build/` directory.
